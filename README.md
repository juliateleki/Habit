# Habit Tracker

A simple habit tracker app for tracking daily habits, building routines, and staying consistent over time.

## Features

- Create, edit, and delete habits
- Mark habits as completed each day
- View daily progress
- Keep a simple record of consistency
- Clean and easy to use interface

## Purpose

The goal of this project is to help users stay accountable to their routines by giving them a simple way to track habits in one place.

## Tech Stack

- Language: Swift
- UI Framework: SwiftUI
- Data Persistence: SwiftData
- Architecture: MVVM
- Charts and progress visuals: Swift Charts
- Local notifications: UserNotifications
- Version control: Git + GitHub
- IDE: Xcode

## Getting Started

1. Clone the repository
2. Install dependencies
3. Start the development server

```bash
git clone [your-repo-link]
cd [your-project-name]
npm install
npm run dev
```
